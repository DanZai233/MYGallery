### 🎨 前台展示（纯展示，无管理入口）

2. ✅ **分类筛选** - 毛玻璃按钮，点击筛选
3. ✅ **照片搜索** - 点击🔍展开，搜索多个字段
4. ✅ **Hover 简介** - 鼠标悬停底部滑入
5. ✅ **灯箱预览** - 点击查看大图
6. ✅ **元数据显示** - 标题居中，参数靠右
7. ✅ **黑夜模式** - 点击🌙切换主题
8. ✅ **键盘导航** - ← → ESC 操作
9. ✅ **图片缩放** - 灯箱内缩放
10. ✅ **响应式设计** - 完美适配所有设备

### ⚙️ 后台管理（仅 URL 访问）

1. ✅ **用户登录** - JWT Token 认证
2. ✅ **照片上传** - 拖拽上传、批量处理
3. ✅ **EXIF 提取** - 自动提取相机元数据
4. ✅ **照片编辑** - 标题/描述/标签/位置
5. ✅ **分类设置** - 为照片设置分类
6. ✅ **EXIF 编辑** - 手动修改相机参数
7. ✅ **照片删除** - 删除照片和文件
8. ✅ **分类管理** - 创建/编辑/删除分类
9. ✅ **网站设置** - 自定义网站信息/备案
10. ✅ **统计面板** - 照片数/浏览量/存储

### 🔧 技术特性

1. ✅ **Golang 后端** - 高性能 Web 服务
2. ✅ **多数据库** - SQLite/MySQL/PostgreSQL
3. ✅ **多存储** - Local/S3/MinIO/阿里云OSS
4. ✅ **JWT 认证** - 安全的身份验证
5. ✅ **EXIF 解析** - 完整的元数据提取

## 🚀 使用流程

### 1. 启动应用

```bash
cd /root/MYGallery
go run main.go
```

### 2. 创建分类

```bash
# 快速创建测试分类
bash scripts/create-test-data.sh
```

### 3. 上传照片

访问 http://localhost:8080/admin
- 拖拽上传照片
- 编辑信息和分类
- 保存

### 4. 前台查看

访问 http://localhost:8080
- 查看照片展示
- 测试分类筛选
- 测试搜索功能
- 切换黑夜模式
